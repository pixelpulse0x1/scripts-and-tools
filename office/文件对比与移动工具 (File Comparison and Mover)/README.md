# 文件对比与移动工具 (File Comparison and Mover)

这是一个 Python 脚本，旨在帮助用户自动化地对比两个文件夹中的文件，并根据文件名和文件大小的对比结果，将源文件夹中的文件智能地分类移动到预设的三个不同文件夹中。

## ✨ 功能特性

- **交互式路径输入**: 无需修改代码，通过命令行交互提示输入源文件夹和目标文件夹路径。
- **安全确认机制**: 在执行任何文件操作前，会显示用户所选路径和即将进行的操作，并要求用户二次确认。
- **智能分类**: 根据以下规则自动对源文件夹的文件进行分类：
  1. **名称和大小均相同**: 文件将被移动到 `./疑似完全相同` 文件夹。
  2. **名称相同但大小不同**: 文件将被移动到 `./名称相同大小不同` 文件夹。
  3. **名称不重复**: 文件将被移动到 `./疑似完全不同` 文件夹。
- **自动创建文件夹**: 脚本会自动创建上述三个用于分类的文件夹，无需手动操作。
- **清晰的操作日志**: 在移动每一个文件时，都会在控制台打印清晰的日志，方便追踪操作过程。
- **高兼容性**: 仅使用 Python 内置库 (`os`, `shutil`)，无需安装任何第三方依赖，完美兼容 Windows, macOS 和 Linux。
- **健壮性**: 包含路径有效性检查和基本的错误处理。

## 🚀 如何使用

### 先决条件

- 您需要安装 Python 3.x。

### 操作步骤

1. **下载脚本**

   - 将 `file_comparator.py` 文件保存在您电脑的任意位置。

2. **打开终端**

   - **Windows**: 打开 “命令提示符 (CMD)” 或 “PowerShell”。
   - **macOS/Linux**: 打开 “终端 (Terminal)”。

3. **进入脚本目录**

   - 使用 `cd` 命令切换到您保存 `file_comparator.py` 文件的目录。例如：

     ```
     cd C:\Users\YourUser\Desktop\MyScripts
     ```

4. **运行脚本**

   - 在终端中执行以下命令：

     ```
     python file_comparator.py
     ```

5. **按照提示操作**

   - **输入路径**: 脚本会首先提示您输入`源文件夹`的完整路径，按回车键后，再输入`目标文件夹`的路径。

     > **小技巧**: 您可以直接将文件夹从文件管理器中拖拽到终端窗口来自动填充路径。

   - **二次确认**: 脚本会显示您输入的路径和将要执行的操作摘要。请仔细核对，然后输入 `y` 并按回车键开始执行，或输入 `n` 取消操作。

6. **查看结果**

   - 脚本执行完毕后，源文件夹中的文件将被移动到与脚本同目录下的三个新文件夹中：`./疑似完全相同`、`./名称相同大小不同` 和 `./疑似完全不同`。

## 📁 输出文件夹详解

- `疑似完全相同`:
  - **含义**: 源文件夹中的这个文件，在目标文件夹里找到了一个**同名且大小完全一样**的文件。
  - **使用场景**: 这通常意味着文件是重复的，可以考虑删除以节省空间。
- `名称相同大小不同`:
  - **含义**: 源文件夹中的这个文件，在目标文件夹里找到了一个**同名但大小不同**的文件。
  - **使用场景**: 这可能意味着文件是不同版本（例如，修改过的文档、不同质量的图片/音频）。您可能需要手动检查以确定保留哪一个版本。
- `疑似完全不同`:
  - **含义**: 遍历目标文件夹后，没有找到任何与源文件夹中这个文件同名的文件。
  - **使用场景**: 这通常意味着这是源文件夹独有的文件。
