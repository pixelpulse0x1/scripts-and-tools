# Windows 批处理脚本

本目录存放用于 Windows 系统的批处理脚本（.bat/.cmd）。

## 脚本列表

| 脚本文件                                | 用途                                                         | 使用方式 |
| --------------------------------------- | ------------------------------------------------------------ | -------- |
| 重启文件资源管理器.bat                  | Windows文件资源管理器崩溃时专用                              | 双击     |
| 自动刷新DNS.bat                         | 设备频繁切换网络环境，或网络异常时使用                       | 双击     |
| 固定3层目录树生成工具 (最终版)V2.1 .bat | 用于替代没有安装Git bash的电脑使用，模拟tree -d -N -L 3 命令，常用于多硬盘目录查看和管理（给小白使用） | 双击     |

## 注意事项
- 部分脚本需要**管理员权限**。
- 运行前建议先关闭正在编辑的文档，以防重启资源管理器导致未保存丢失。
